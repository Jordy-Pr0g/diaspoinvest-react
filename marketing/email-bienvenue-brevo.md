## Configuration dans Brevo (pas à pas)

1. Menu gauche → **Automatisations** → "Créer une automatisation"
2. Choisir le modèle **"Message de bienvenue"** (Welcome email)
3. Déclencheur : **"Un contact est ajouté à une liste"** → choisir la liste *Newsletter DiaspoInvest*
4. Délai : **immédiat** (ou 5 minutes)
5. Créer l'email avec l'éditeur : coller l'objet, le pré-header et le corps ci-dessus
   - Expéditeur : DiaspoInvest · contact@diaspoinvest.fr
   - La balise de désinscription est ajoutée automatiquement par Brevo dans le footer
6. **Activer** l'automatisation (bouton en haut à droite)
7. Tester : inscris une adresse de test sur la landing → l'email doit arriver dans les minutes qui suivent
